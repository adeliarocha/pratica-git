comando pwd (enter) nos diz em que pasta estamos
comando ls (enter) nos diz o que esta na pasta
comando cd "nome da pasta" (enter) para ir para outra pasta
comando cd .. (enter) para voltar a primeira pasta acessada
comando mkdir "nome da pasta" (enter) para criar uma nova pagina
comando cat "nome do arquivo" (enter) para ler o conteudo do arquivo localizado na pasta
comando mkdir repo (enter) para criar o repositório
comando cd repo (enter) para abrir a pasta repo
comando git init --bare (enter) para inicializar o repositório
comando cd .. para voltar 
comando git clone repo pratica-git-nome para criar uma pasta sua
comando cd pratica-git-nome para entrar na pasta
comando git config user.name "nome" 
comando git config user.email "email"
